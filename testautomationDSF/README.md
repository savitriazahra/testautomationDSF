# testautomationDSF
Test technical automation DSF
**Step 1** -> Download phyton jika belum ada di https://www.python.org/downloads/windows/
**Step 2** Configure Chromedriver -> Download chromedriver (Selenium) 
Cek versi Chrome -> Chrome → Settings → About Chrome
Download ChromeDriver yang sesuai versi Chrome: https://chromedriver.chromium.org/downloads
**Step 3** Running the automation script 
running code menggunakan -> python main.py [Main.py diganti menyesuaikan nama file script yang dibuat yaitu test_login.py]
**Step 4** -> Check report automation test
pastikan Plugin pytest-html sudah terinstall, jika sudah running pytest --html=report.html
jika sudah dirunning maka akan muncul file report.html didalam folder test, lalu double klik dan akan terbuka di browser chrome/Edge
